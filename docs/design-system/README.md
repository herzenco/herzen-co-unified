# Herzen Co. logo system

The governing reference is [Herzen-Co-logo-system.pdf](./Herzen-Co-logo-system.pdf).
The production PNGs in `assets/brand/` come from the official export archive,
[Logos-Herzen-Co.zip](./Logos-Herzen-Co.zip).

## Approved assets

- `logo-1a-black.png` - primary horizontal lockup on light backgrounds
- `logo-1a-white.png` - primary horizontal lockup on ink or dark backgrounds
- `logo-1e-black.png` - plated lockup on light backgrounds
- `logo-1e-white.png` - plated lockup on ink or dark backgrounds
- `mark-h-black.png` - isolated H mark on light backgrounds
- `mark-h-white.png` - isolated H mark on dark backgrounds
- `wordmark-black.png` - isolated wordmark on light backgrounds
- `wordmark-white.png` - isolated wordmark on dark backgrounds
- `icon.png` - primary 512px app icon and favicon
- `icon-mark.png` - transparent 512px icon mark
- `logo-black.png` - original vertical lockup on light backgrounds
- `logo-white.png` - original vertical lockup on dark backgrounds

## Usage rules

- The website uses horizontal `logo-1a` lockups exclusively.
- Keep the original vertical `logo-black` and `logo-white` files archived; do not render them on the website.
- Use the primary horizontal lockup for website headers, documents, and signatures.
- Use the reversed white lockup for footers, covers, and dark sections.
- Keep at least the mark's crossbar-height of clear space on every side.
- Keep the full lockup at or above approximately 24px tall.
- Below 24px, use an isolated H mark or the purpose-built icon export.
- Do not redraw, stretch, recolor, crop, or recombine the approved artwork.
